## Selected work

Most of the work below is private client engineering, so the source is not public. Short case studies:

**LISRMA — Living Evidence Synthesis Platform** *(Mayo Clinic)*
Automated living-systematic-review platform with multi-agent title/abstract screening and structured extraction over 35,000–40,000 papers per review.
- Reduced manual literature-screening effort by ~70%
- ~93% screening agreement with human reviewers on a held-out sample
- Retrieval recall@10 ~88%; field-level extraction accuracy ~90%
- Claude Agent SDK, LangGraph, RAG, Qdrant, FastAPI, Celery, Redis, Docker

**TAX GENII — Retrieval-Augmented Chatbot for Australian Tax Law**
Natural-language querying over 200,000+ tax-law documents using semantic and vector retrieval.
- ~90% answer relevance; under 4s average latency on multi-turn queries
- Azure AI Search, Flask, Python, SQL

**Al Jazeera Finance Chatbot**
Conversational financial assistant combining structured JSON question flows with an LLM fallback that preserves full conversation context.
- Resolved the majority of queries without escalation; context-aware multi-turn answers over Al Jazeera content
- LLMs, RAG, FastAPI, MySQL

**Synnc — Collaboration Platform for Influencers and Brands**
Campaign-lifecycle platform with multi-tenant RBAC and AI-assisted copywriting.
- End-to-end campaign management for ~50 brands and influencers; ~60% reduction in content-drafting time
- FastAPI, Firebase, OpenAI API, MongoDB, Docker, S3
